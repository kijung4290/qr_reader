> 카메라가 정상 작동하도록 만든 독립 스캐너예요.
>
> 코딩 몰라도 괜찮아요! 클릭만 따라하면 **10분이면 끝**입니다 ☕

## 📋 STEP 1. GAS 코드 업데이트하기

> ⚠️ **이 단계를 꼭 먼저 해야 QR 스캐너가 출석 처리를 할 수 있어요!**

### 1-1. Apps Script 열기

1. 기존 **경로식당 출석 스프레드시트** 열기
2. 상단 메뉴 → **확장 프로그램** → **Apps Script** 클릭

### 1-2. Code.gs 코드 교체

1. 왼쪽에서 **Code.gs** 파일 클릭
2. 기존 코드를 **전체 선택** (`Ctrl+A`)
3. 이 프로젝트의 **Code.gs** 파일 내용을 **통째로 복사**해서 **붙여넣기** (`Ctrl+V`)
4. 💾 **저장** (`Ctrl+S`)

### 1-3. 웹앱 재배포 (기존 배포 업데이트)

> 💡 **"새 배포"가 아니라 "기존 배포 업데이트"**를 해야 기존 QR코드들이 계속 작동해요!

1. 상단 **배포** → **배포 관리** 클릭
2. 오른쪽 위 **✏️ 연필 아이콘** (수정) 클릭
3. **버전** → **새 버전** 선택
4. **배포** 버튼 클릭
5. 나오는 **웹앱 URL을 복사**해서 메모장에 저장해두세요!

| 항목 | 이렇게 설정하세요 |
|------|------------------|
| 설명 | `QR 스캐너 JSONP 지원 추가` |
| 실행 사용자 | **나** |
| 액세스 권한 | **모든 사용자** |
| 버전 | **새 버전** |

> ✅ 여기서 복사한 URL이 나중에 STEP 4에서 필요해요! 꼭 메모해두세요!

## 📋 STEP 2. GitHub에 파일 올리기

> 💡 GitHub 계정이 없으면 먼저 [github.com](https://github.com) 에서 무료로 가입하세요!

### 2-1. 새 저장소 만들기

1. [github.com](https://github.com) 로그인
2. 오른쪽 위 **+** 버튼 → **New repository** 클릭
3. 아래처럼 설정하세요:

| 항목 | 이렇게 입력하세요 |
|------|------------------|
| Repository name | `qr-scanner` |
| Description | `경로식당 QR 출석 스캐너` |
| Public / Private | **Public** (공개) 선택 |
| Add a README file | ✅ 체크하지 마세요 (비워두세요) |

4. **Create repository** 초록 버튼 클릭

### 2-2. 파일 업로드하기

> 저장소가 만들어지면 빈 페이지가 나와요. 거기서 파일을 올릴 거예요!

1. 화면에 보이는 **"uploading an existing file"** 링크를 클릭
   - (또는 상단 **Add file** → **Upload files** 클릭)
2. `qr-scanner` 폴더 안에 있는 **2개 파일**을 선택해서 드래그 앤 드롭:
   - 📄 `index.html`
   - 📄 `vercel.json`
3. 아래쪽 **Commit changes** 초록 버튼 클릭

> ✅ 이렇게 하면 GitHub에 파일이 올라간 거예요! 쉽죠? 😊

## 📋 STEP 3. Vercel에서 배포하기

> 💡 Vercel 계정이 없으면 [vercel.com](https://vercel.com) 에서 **Continue with GitHub** 로 간편 가입하세요!

### 3-1. Vercel에 GitHub 연결

1. [vercel.com](https://vercel.com) 로그인
2. 대시보드에서 **Add New...** → **Project** 클릭
3. **Import Git Repository** 에서 방금 만든 `qr-scanner` 저장소를 찾아 **Import** 클릭

> 💡 저장소 목록에 안 보이면? → **"Adjust GitHub App Permissions"** 클릭 → 저장소 권한을 추가해주세요!

### 3-2. 프로젝트 설정

| 항목 | 이렇게 설정하세요 | 왜? |
|------|------------------|-----|
| Project Name | `qr-scanner` (자동입력됨) | 배포 URL이 됩니다 |
| Framework Preset | **Other** | 순수 HTML 프로젝트 |
| Root Directory | `.` (기본값 그대로) | 변경 불필요 |
| Build Command | **비워두세요** | 빌드 없이 바로 배포 |
| Output Directory | `.` (기본값 그대로) | 변경 불필요 |

### 3-3. 배포 실행

1. **Deploy** 버튼 클릭
2. 30초~1분 기다리면 **🎉 Congratulations!** 화면이 나와요
3. 제공된 URL이 바로 QR 스캐너 주소예요!

> ✨ URL 형태: `https://qr-scanner-xxxxx.vercel.app`

---

## 📋 STEP 4. QR 스캐너에 웹앱 URL 설정하기

> 마지막 한 단계만 하면 끝이에요!

1. STEP 3에서 받은 **Vercel 배포 URL**로 접속
2. 오른쪽 위 **⚙️ 설정** 버튼 클릭
3. **Google Apps Script 웹앱 URL** 입력란에 **STEP 1에서 복사한 URL** 붙여넣기
4. **저장** 클릭

> ✅ 완료!! 이제 **카메라 시작** 버튼을 누르고 QR코드를 비춰보세요! 🎉

## 🎊 사용 방법

### 기본 출석 (식당 입구 태블릿)

1. Vercel 배포 URL을 태블릿 브라우저에서 열기
2. **"카메라 시작"** 버튼 클릭
3. 카메라 권한 **허용**
4. 어르신 QR코드를 카메라에 비추기
5. ✅ **"출석 완료"** 확인
6. 3초 후 자동으로 다음 분 스캔 대기

### 꿀팁 🍯
- 홈 화면에 추가하면 앱처럼 사용할 수 있어요! (브라우저 메뉴 → "홈 화면에 추가")
- 스캔 기록은 당일만 유지돼요 (브라우저를 닫으면 초기화)
- 설정(URL, 대기시간)은 저장되므로 매번 입력 안 해도 돼요

## ❓ 자주 묻는 질문 (FAQ)

### Q. 📱 기존 출석 앱은 계속 사용할 수 있나요?

> **A.** 네, 100% 사용 가능해요! 기존 앱은 그대로 유지되고, QR 스캐너만 별도로 추가되는 거예요.
> 이름 검색 출석은 기존 앱에서, QR 스캔 출석은 이 스캐너에서 하시면 됩니다.

### Q. 💰 비용이 드나요?

> **A.** 아뇨, **완전 무료**예요! GitHub도 Vercel도 이 정도 규모는 무료로 사용 가능해요.

### Q. 📡 인터넷이 안 되면 어떻게 되나요?

> **A.** 인터넷 연결이 필요해요. QR 스캔 후 구글 서버에 출석을 기록하는 방식이라,
> 인터넷이 안 되면 "서버에 연결할 수 없습니다" 오류가 나옵니다.

### Q. 🔐 보안은 괜찮은가요?

> **A.** 네, 안전해요! 데이터는 본인의 구글 스프레드시트에만 저장되고,
> Vercel 앱은 QR을 읽고 구글에 출석 요청만 보내는 역할이에요.

### Q. 😰 "서버 응답 시간 초과" 오류가 나와요

> **A.** STEP 1의 GAS 코드 업데이트를 안 하셨거나, 배포를 안 하신 거예요.
> STEP 1을 다시 확인해주세요!

### Q. 📷 카메라가 안 열려요

> **A.** 브라우저 주소가 `https://`로 시작하는지 확인하세요.
> 카메라는 보안 연결(HTTPS)에서만 작동해요. Vercel 배포 시 자동으로 HTTPS가 적용됩니다.
